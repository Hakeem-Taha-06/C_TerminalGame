Terminal-based C game that involves:
-  Simple WASD controls for player character movement.
-  Triggering switches to unlock an exit.
-  BFS path finding algorithm for enemy AI.

Controls:
-  Type one of the four direction letters "WASD" in the terminal then press enter to register the input.

Goal:
-  Activate all the switches "o" then reach the exit "+" to win.
-  If you get hit by the enemies "<", ">", "v", "^", "X", you lose.

How to run:
-  clone the repo with
```

git clone https://github.com/Hakeem-Taha-06/C_TerminalGame

```
-  run the app.
-  type the level file name from the existing files (OR MAKE YOUR OWN, it's really simple).
-  press enter.
